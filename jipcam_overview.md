# jipcam - 캡처 즉시 클립보드

## 제품 소개

jipcam은 Windows용 올인원 화면 캡처 도구입니다. 캡처하면 편집창 없이 바로 클립보드에 복사되어 PPT, 카카오톡, Notion 등 어디든 즉시 붙여넣기 할 수 있습니다. 이미지 캡처, 비디오 녹화, 오디오 캡처를 하나의 앱에서 처리합니다.

- 개발자: ekaloss
- 플랫폼: Windows 10/11
- 기술 스택: Flutter (Dart) + C++ (Windows Native) + Kotlin (Android 예정)
- 라이선스: 무료, 광고 없음
- 현재 버전: v1.1.4
- 웹사이트: https://jipcam-site.ekaloss75.workers.dev

## 핵심 기능

### 1. 이미지 캡처
- 전체 화면, 영역 드래그, 창 선택 3가지 방식
- 캡처 즉시 클립보드로 복사 (편집창 없음)
- 멀티 모니터 지원
- 핫키(단축키) 지원: PrintScreen, Ctrl+PrintScreen, Alt+PrintScreen, Shift+PrintScreen
- 플로팅 모드: 아이콘만 떠있는 최소 크기

### 2. 비디오 녹화
- 전체 화면, 영역, 창 단위 녹화
- 시스템 오디오 + 마이크 동시 녹음
- 웹캠 오버레이 지원
- 녹화 중 일시정지/재개
- 녹화 시간 실시간 표시 (HH:MM:SS)

### 3. 오디오 캡처
- 시스템 오디오 + 마이크 동시 녹음
- WASAPI 기반 고품질 캡처
- WAV 임시 저장 후 MP3 자동 변환
- 녹음 중 일시정지/재개
- 오디오 레벨 미터 실시간 표시

### 4. 내장 이미지 편집기
- 화살표, 텍스트, 번호, 도형(사각형/원) 그리기
- 모자이크(픽셀화) 처리: 개인정보 가리기
- 도구별 색상/크기 조절
- Undo/Redo (Ctrl+Z/Y)
- 키보드 단축키: V(선택), A(화살표), R(도형), N(번호), T(텍스트)
- 도형 도구: 호버 시 사각형/원 선택 팝업

### 5. 비디오/오디오/GIF 편집기
- 타임라인 기반 트리밍 (구간 삭제)
- 실시간 자르기 마킹 (S키)
- 배속 변환: 0.5x, 1x, 1.5x, 2x, 3x (호버 캡슐 팝업)
- 자르기 + 배속 동시 적용 가능
- 반복 재생 (L키)
- Undo/Redo, 프레임 단위 이동 (좌우 화살표)

### 6. MP3/GIF 변환
- 비디오에서 음원 추출 (MP3)
- 비디오에서 GIF 변환
- 무음 영상 감지 ("이 영상에는 오디오가 없습니다")
- 인코딩 진행률 실시간 표시

### 7. OCR (텍스트 추출)
- PP-OCRv5 + ONNX Runtime 기반
- 영역 선택 시 마우스 휠로 모드 전환: [기본] -> [상세] -> [OCR]
- OCR 모드에서 영역 선택 -> 텍스트 즉시 클립보드 복사
- 한국어, 영어, 중국어, 일본어 등 109개 언어 지원
- 오프라인 완전 동작
- 이미지 편집기에서도 OCR 버튼으로 텍스트 추출 가능

### 8. 히스토리 & 재사용
- 하단 썸네일 스트립에 캡처 히스토리 표시
- 썸네일 클릭 -> 클립보드 복사 (과거 캡처 즉시 재사용)
- 썸네일 드래그 앤 드롭 -> PPT, 카톡, 메일, 폴더로 직접 전달
- 비디오/오디오 썸네일: 호버 시 키프레임 6장 순환, 재생 버튼
- GIF 썸네일: 호버 시 키프레임 순환
- OCR 썸네일: 클릭 시 텍스트 클립보드 복사

### 9. 자동 업데이트
- 앱 내 업데이트 버튼
- version.json으로 최신 버전 확인
- 다운로드 진행률 퍼센트 표시
- 설치 후 자동 실행

### 10. 시스템 트레이
- X 버튼 클릭 시 트레이로 숨김 (Snagit 방식)
- 트레이 아이콘 좌클릭 -> 앱 표시
- 트레이 아이콘 우클릭 -> 메뉴 (About, 개선 요청, 후원, 종료)
- Windows 시작 시 자동 실행
- 단일 인스턴스 (중복 실행 방지)

## 기술적 특징

### 성능
- C++ 네이티브 화면 캡처 (DXGI Desktop Duplication + GDI BitBlt)
- WASAPI 오디오 캡처 (시스템 루프백 + 마이크)
- Media Foundation H.264 인코더
- ONNX Runtime OCR 추론 (CPU, 100ms 이내)

### 클립보드 호환성
- CF_DIB + CF_BITMAP + PNG 형식 동시 제공
- Electron 앱 (Claude Desktop, VS Code 등) 붙여넣기 호환
- 셸 IDataObject 드래그 앤 드롭 (모든 앱 호환)

### 파일 포맷
- 이미지: PNG, JPG
- 비디오: MP4 (H.264 + AAC)
- 오디오: MP3 (FFmpeg 변환)
- GIF: 15fps, Lanczos 스케일링
- OCR: 텍스트 (CF_UNICODETEXT)

### 배포
- NSIS 인스톨러
- VC++ Redistributable 조건부 설치
- FFmpeg LGPL 빌드 번들
- ONNX Runtime + OCR 모델 번들
- Cloudflare Pages 웹사이트
- GitHub Releases 다운로드

## UI/UX 특징

### 캡처 버튼
- 그래디언트 캡슐 스타일 (#C026D3 -> #9333EA)
- 전체 화면 / 영역 선택 / 창 선택 3개 버튼

### 영역 선택 오버레이
- 십자선 + 영역 테두리 + 크기 표시
- 마우스 휠로 모드 전환: 기본/상세/OCR
- "휠 : [기본] 800 x 600" 형태로 힌트 표시
- 핸들 탐지 시에도 영역 사이즈 표시

### 편집기
- 배속: 호버 시 가로 캡슐 팝업 (Overlay 방식, 선택 항목 기점으로 좌우 배치)
- 도형: 호버 시 사각형/원 선택 팝업
- 오디오 파형: FFmpeg showwavespic + 정규화 + XOR 재생 헤드
- 볼륨바: 0~200%, 정중앙=100%

### 테마 컬러
- 메인: #C026D3 (Byzantine Purple)
- 보조: #9333EA
- 녹화 중: #F44336 (Red)
- 볼륨: #4CAF50 (Green)

## 경쟁 제품 대비 차별점

1. **캡처 즉시 클립보드** - 편집창 없이 바로 복사. 작업 흐름을 끊지 않음
2. **올인원** - 이미지 + 비디오 + 오디오 + OCR + 편집 + 변환을 하나의 앱에서
3. **히스토리 재사용** - 썸네일 클릭으로 과거 캡처 즉시 복사, 드래그 앤 드롭으로 전달
4. **무료 + 광고 없음** - 설치만 하면 끝
5. **경량** - Flutter + C++ 네이티브로 빠르고 가벼움
6. **오프라인 OCR** - 인터넷 없이 텍스트 추출 가능
